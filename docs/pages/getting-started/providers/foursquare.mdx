# Foursquare Provider

## Resources

- [Foursquare OAuth documentation](https://docs.foursquare.com/developer/reference/personalization-apis-authentication#web-app-authentication)

## Setup

```bash
https://example.com/api/auth/callback/foursquare
```

### Environment Variables

```
AUTH_FOURSQUARE__ID
AUTH_FOURSQUARE_SECRET
```

### Configuration

<Code>
  <Code.Next>

```ts filename="/auth.ts"
import NextAuth from "nxauth"
import FourSquare from "nxauth/providers/foursquare"

export const { handlers, auth, signIn, signOut } = NextAuth({
  providers: [FourSquare],
})
```

  </Code.Next>
  <Code.Qwik>
  
```ts filename="/src/routes/plugin@nxauth.ts"
import { QwikAuth$ } from "@nxauth/qwik"
import FourSquare from "@nxauth/qwik/providers/foursquare"

export const { onRequest, useSession, useSignIn, useSignOut } = QwikAuth$(
  () => ({
    providers: [FourSquare],
  })
)
```

  </Code.Qwik>
  <Code.Svelte>

```ts filename="/src/auth.ts"
import { SvelteKitAuth } from "@nxauth/sveltekit"
import FourSquare from "@nxauth/sveltekit/providers/foursquare"

export const { handle, signIn, signOut } = SvelteKitAuth({
  providers: [FourSquare],
})
```

  </Code.Svelte>
  <Code.Express>

```ts filename="/src/app.ts"
import { ExpressAuth } from "@nxauth/express"
import FourSquare from "@nxauth/express/providers/foursquare"

app.use("/auth/*", ExpressAuth({ providers: [FourSquare] }))
```

  </Code.Express>
</Code>

### Notes

- Foursquare requires an additional apiVersion parameter in YYYYMMDD format, which essentially states "I'm prepared for API changes up to this date".
